# Filtering Your Library

The filter panel lets you narrow down books on the **Library**, **All Books**, and similar pages using one or more criteria. Open it by clicking the **Filters** button (shows a badge with the active filter count when any filters are set).

## Filter Mode

A segmented control at the top of the panel sets how multiple filters combine:

| Mode    | Behaviour                                          |
| ------- | -------------------------------------------------- |
| **AND** | A book must match **all** active filters (default) |
| **OR**  | A book must match **at least one** active filter   |
| **NOT** | Books that match **any** active filter are hidden  |

## Status

Filter by your personal read status. Select one or more chips:

- **Unread** — not yet started
- **Reading** — currently in progress
- **Read** — finished
- **Won't Read** — marked as not interested

## Rating

### My Rating (min)

Show only books you have rated at or above a threshold (1–5 stars). Select **Any** to remove the filter.

### Goodreads Rating (min)

Show only books with a Goodreads community rating at or above a threshold (3, 3.5, 4, or 4.5). Requires the book to have a Goodreads rating stored — books without one are excluded when this filter is active.

## Content

These sections only appear when the relevant data exists in your library.

### Genres

Check one or more genres to include. A book matches if it belongs to any of the selected genres.

### Tags

Check one or more tags. A book matches if it has any of the selected tags.

### Moods

Check one or more moods. A book matches if it has any of the selected moods.

## Authors

A searchable, scrollable list of every author in the current view, sorted by book count. Check one or more authors to include only their books.

Use the **Search authors…** box to filter the list by name.

## Book Details

### Format

Filter by file format (EPUB, MOBI, PDF, etc.). A book matches if any of its files are in a selected format.

### Publisher

Check one or more publishers. Only appears when publisher data is present.

### Page Count

Bucket books by length:

| Bucket      | Range                           |
| ----------- | ------------------------------- |
| **< 150**   | Short (novellas, short stories) |
| **150–400** | Medium                          |
| **400+**    | Long                            |

Books without a page count are excluded when this filter is active.

### Published Year

Enter a **From** year, a **To** year, or both to restrict to a publication window. Books without a published date are excluded when this filter is active.

## Library

### Series

| Option         | Behaviour                                |
| -------------- | ---------------------------------------- |
| **Any**        | No series filter (default)               |
| **In Series**  | Only books that belong to a named series |
| **Standalone** | Only books with no series assigned       |

### Added to Library

Restrict to books imported within a recent window:

| Option  | Cutoff                   |
| ------- | ------------------------ |
| **Any** | No date filter (default) |
| **7d**  | Last 7 days              |
| **30d** | Last 30 days             |
| **6mo** | Last 6 months            |

## Clearing Filters

Click **Clear all** in the filter panel header to reset every filter at once (the filter mode is preserved). The active filter count badge disappears when no filters are set.
